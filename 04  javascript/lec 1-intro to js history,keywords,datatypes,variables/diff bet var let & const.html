Diff betn var let & const

  1. let & const are block scoped where as var is function or global scope
  2. let & var can be reassigned, whereas const cannot be reassigned
  3. Redeclaration is possible incase of var keyword whereas redeclaration is not poissble incase of let & const keyword
  4. if we try to access the varbles declared with var keyword before their declaration then these variables values will be undefiend, whereas if we try to access variables declared with let & const before there declaration then it will generate reference Error (because values are present inside TMP - Temporal Dead Zone)
