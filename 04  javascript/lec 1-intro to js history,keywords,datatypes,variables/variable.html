Variables in javascript

We can create variables in 3 ways in javascript

  1. using let keyword
  2. using const keyword
  3. using var keyword

Indentifier Rules

  1. Indetifier cannot be start with or use special characters (only $ , _ )
  2. Indentifier cannot same as Keyword
  3. Indentifier cannot start with any number(but use number)
  4. Indetifier can any character between "A" to "Z" and "a" to "z"
